# Clipboard Utility

A simple Python application that copies text to the system clipboard and retrieves it using the Pyperclip library.

## Features

- Copy text to the clipboard
- Retrieve clipboard contents
- Beginner-friendly Python project

## Technologies Used

- Python
- Pyperclip

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the program:

```bash
python clipboard_utility.py
```

Enter any text when prompted, and the program will copy it to your clipboard and display the copied text.

## Requirements

- Python 3.x
- Pyperclip
